Background music folder
========================
Drop any audio file here: WAV, MP3, AAC, OGG, FLAC.
The script picks the first audio file it finds.
A generated ambient pad (background.wav) is created automatically
if no file exists. Replace it with any royalty-free track.

Good sources of CC-licensed music:
  • https://freemusicarchive.org
  • https://ccmixter.org
  • https://freesound.org
